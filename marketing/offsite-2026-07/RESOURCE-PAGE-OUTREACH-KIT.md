# Resource-Page Outreach Kit — manual, no tools required

**What this is for:** the blueprint's "Resource Page Link Capture" strategy —
finding human-curated lists of marketing/AI tools and pitching to be added.
This is legitimate, standard outreach (real value, real human editor,
real opt-in inclusion) as long as every pitch is genuinely relevant to the
page it's sent to — this stops being legitimate the moment it becomes
mass, irrelevant, or automated blasting. Treat every send as one real email
to one real person, per `rules/content-integrity.md`.

**Action needed from Constantin:** run the searches below, build a short list
of real target pages, and send the pitch email (adapted per page) manually.
This is not something to automate or batch-send — see the "don't" note at
the end.

---

```
"marketing tools" AND "helpful resources"
"AI tools directory" AND "submit"
inurl:resources "artificial intelligence marketing"
"best AI tools for marketers" -site:getbrandgeo.com

```

**Given the Search Console signal (US law firms searching for AI visibility

```
"legal marketing tools" AND "resources"
"law firm marketing" "recommended tools" OR "tool list"
inurl:resources "legal marketing" OR "law firm SEO"
"legal tech directory" submit
"attorney marketing blog" "tools we recommend"
```

## Part 2 — How to evaluate a candidate page before pitching

Only pitch a page if, honestly:
- It's a real, human-curated list (not an auto-generated scraped directory
  with hundreds of unrelated entries).
- BrandGEO would be a genuinely useful addition for that page's actual
  audience — not just "any resource page that'll take a link."
- The page is reasonably current (a resources page last updated in 2021 is
  unlikely to be maintained by anyone actively reviewing submissions).

## What NOT to do (this is what keeps this legitimate)

- Don't send the same email verbatim to 50 pages — every send should
  reference something real and specific about that page.
- Don't pitch a page that isn't a genuine fit just because it'll take a link.
- Don't automate this with a bulk-outreach tool — this is the one part of
  the whole blueprint that's explicitly manual, and that's the point: a
  human editor choosing to include you because it's genuinely useful is
  what makes the resulting link real, not the volume of emails sent.
- If a page asks for payment to be listed, that's a paid link, not organic
  outreach — decline, consistent with this project's standing
  content-integrity rule.

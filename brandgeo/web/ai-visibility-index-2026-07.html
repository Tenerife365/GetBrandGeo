The AI Visibility Index: July 2026

This is the first issue of a monthly report built entirely from BrandGEO's own collection data: real commercial-buyer prompts run against real AI engines, then compared side by side. No survey data, no vendor estimates, no simulated queries.

It's also, honestly, an early one. This issue leans on a 7-city research program that ran a single collection pass per city, not a long-running trend. We're publishing it anyway, with the limits stated plainly, because a caveated real number is worth more than a polished guess.

The pattern that held in every city we tested

Across London, Berlin, Madrid, New York, Paris, Rome, and Dublin, the same split showed up again and again: categories with a small number of well-documented, frequently-reviewed products get near-unanimous AI recommendations, while categories built on a long tail of small local firms — law, independent financial advice, boutique real estate — stay fragmented, with every engine naming a different set of names.

The sellable insight, in one line

Software and fintech categories are already decided — AI engines agree with each other because there's a small set of well-documented products to agree on. Professional services with thousands of thinly-documented local firms are still wide open, in every city we've checked so far.

Rome: a genuinely different pattern, worth flagging on its own

Every other city in this program had at least one category with a clean cross-engine winner. Rome didn't. Instead, the striking result was within-engine stability: Meta AI reproduced a word-for-word identical, identically-ordered five-restaurant list in both Italian and English for the same prompt, and Perplexity named essentially the same nine real-estate agencies in both languages, just reordered. No brand in Rome crossed the 3-of-4-engine consensus bar any category in London, Paris, Berlin, New York, or Dublin managed. Rome's AI Visibility landscape currently reads less like "the engines agree" and more like "each engine has built its own stable, internally consistent worldview that doesn't transfer to the others."

Language doesn't just change the ranking, it can change the category

The single most consequential finding in this issue's underlying research came from Paris. Asking for wealth management advisors in French surfaced boutique, independent French patrimoine firms — Cheval Blanc Patrimoine appeared in 3 of 4 French-language responses and in zero English-language ones. Asking the identical question in English surfaced major international private banks instead — Société Générale Private Banking and BNP Paribas Wealth Management, both effectively unanimous once the English runs are combined. This isn't the same information reordered by language. It's a different competitive set entirely, depending on which language the question was asked in. A firm that only publishes in French is structurally invisible to English-language AI searchers in this category, by design, not by accident.

Being honest about this issue's limits

This is one snapshot, not a trend. Every city above reflects a single collection run on 2026-07-10. We don't yet know if these consensus patterns hold a month from now, and Issue #2 is where we'll find out.

ChatGPT collection failed on every prompt in every city this run. That's a documented collection error on our side, not a real "ChatGPT recommends nothing" signal — the findings above are drawn from Claude, Gemini, Meta AI, and Perplexity, and the ChatGPT gap is being fixed.

Real, paying-client data only started accumulating daily as of 2026-07-13. Until then, this dataset was blocked on access to enough live volume to be worth reporting on. Issue #2 onward will be able to show actual month-over-month movement from real accounts, not just single-city snapshots.

● Common Questions

Is the AI Visibility Index based on real data?

Yes. Every number in this issue comes from BrandGEO's own collection pipeline: real commercial-buyer prompts run against real AI engines (ChatGPT, Gemini, Claude, Perplexity, and Meta AI), with the responses read and compared directly. Nothing is estimated or simulated.

Why publish Issue #1 now instead of waiting for more data?

Because the alternative is a marketing claim instead of a data point. This issue is honest about being a single snapshot rather than a longitudinal trend, and about a real ChatGPT collection gap in the underlying research. Publishing now, caveated, is more useful than waiting for a bigger sample and calling it certainty.

How often will the AI Visibility Index publish?

Monthly. Real client collection data has been accumulating daily since 2026-07-13, so future issues will add trend lines this first issue can't show yet, on top of new research.
